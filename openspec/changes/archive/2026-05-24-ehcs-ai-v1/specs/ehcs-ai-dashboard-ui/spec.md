# ehcs-ai-dashboard-ui Specification

> **Superseded for implementation detail by** `openspec/specs/ehcs-ai-dashboard-ui/spec.md` **(V1.1.1).** Archived copy retained for change history.

## Purpose

Dashboard and global EHCS layout per PRD V1.1.1.

## Requirements

### Requirement: Full-width dashboard

The dashboard SHALL use full-width stats, trend, and recent anomalies (no permanent 70% right chat column).

### Requirement: Batch check via agent dock

Full check SHALL be triggered from the agent dock「开始检查」, not from a table-header button.
